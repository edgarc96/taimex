TAIMEX - WIP Page Generator Tool

How to Use This Tool

  1. This is a client-side tool for generating WIP pages. Download this file to your local computer and open it in a web browser.
  2. Select a template type from the options below.
  3. Fill in the page details and add your BASIC code and hidden fields.
  4. Click "Generate Page" to create your WIP page code.
  5. Click "Copy Code" and paste the code into a new .HTM file on the server.
